계정 및 데이터 삭제 안내

📌 BakeLog 계정 삭제 방법

개발자: Luna · 문의: support@bakelog.app

요약

  • 앱 내 경로: 설정 → 계정 관리 → 계정 삭제
  • 또는 이메일/문의 양식으로 삭제 요청 가능
  • 삭제 대상: 계정 식별자, 인증 토큰, 프로필/기본 설정, 레시피 등 사용자 생성 데이터
  • 보관 예외: 법적 의무 또는 분쟁 처리 목적의 최소 정보는 최대 30일 간 보관 후 자동 삭제

앱 내에서 직접 삭제

  1. 설정 → 계정 관리로 이동
  2. 계정 삭제 선택 및 확인
  3. 앱 재시작 후 로그아웃 상태 확인

직접 삭제가 보이지 않을 경우 아래의 삭제 요청 방법을 이용해 주세요.

삭제 요청 (대체 경로)

다음 방법 중 하나로 요청해 주세요.

  • 이메일: support@bakelog.app
  • 문의 양식: https://forms.gle/your-form-id (옵션, 폼 사용 시 링크 교체)

요청 시 다음 정보를 포함해 주세요: 가입 이메일(또는 Google UID), 앱 버전, 기기 모델.

삭제/보관되는 데이터

삭제되는 데이터

  • 계정 식별자: 익명 UID, 이메일 기반 UID, Google 로그인 UID
  • 인증 토큰 및 연결 정보 (Firebase Authentication)
  • 프로필/기본 설정(앱 내 저장)
  • 사용자 생성 콘텐츠(예: 레시피, 메모 등)

보관/예외

  • 법적 의무 이행, 분쟁/사기 방지를 위한 최소 정보는 최대 30일 보관 후 파기
  • 백업/로그 시스템에 남을 수 있는 기록은 순환 보관 주기에 따라 제거
  • 카메라 촬영 이미지는 서버에 지속 보관하지 않음 (AI 분석 후 폐기)

Firebase(인증/스토리지/데이터베이스)를 사용하며, 데이터는 전송 시 암호화됩니다.

계정 유형별 안내

  • 익명 계정: 삭제 시 연결된 UID 및 관련 데이터가 즉시 삭제됩니다.
  • 이메일/비밀번호: 본인 확인 후 계정 및 데이터가 삭제됩니다.
  • Google 로그인: 요청 시 해당 Google UID에 연결된 데이터가 삭제됩니다. (Google 계정 자체는 영향을 받지 않음)

처리 기한 및 결과 통지

  • 요청 접수 후 7일 이내 처리
  • 처리 완료 시 이메일로 결과 안내
  • 본인 확인이 불가한 경우 추가 정보 요청 가능

문의

데이터 삭제와 관련한 문의는 support@bakelog.app 로 연락해 주세요.

본 안내는 Google Play 데이터 삭제 정책을 준수하기 위한 것이며, 관련 법령(예: 전자상거래법, 세법 등)에 따라 일부 데이터는 보관 의무가 있을 수 있습니다.
